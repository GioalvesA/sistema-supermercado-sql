# Sistema Supermercado SQL

Projeto acadêmico de Banco de Dados desenvolvido utilizando MySQL.

## 📌 Objetivo
Simular o funcionamento de um sistema de supermercado com:
- Clientes
- Produtos
- Funcionários
- Vendas
- Estoque

## 🛠 Tecnologias
- MySQL
- SQL
- Modelagem de Dados

## 📂 Estrutura do Projeto
- Scripts SQL
- Modelo DER
- Relacionamentos
- Consultas SQL

## 🚀 Aprendizados
Neste projeto pratiquei:
- CREATE TABLE
- FOREIGN KEY
- INNER JOIN
- SELECT
- INSERT
- UPDATE
- Modelagem relacional
